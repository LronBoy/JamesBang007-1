鼠标事件

https://developer.mozilla.org/zh-CN/docs/Web/API/MouseEvent

MouseEvent 接口指用户与指针设备( 如鼠标 )交互时发生的事件。使用此接口的常见事件包括：click，dblclick，mouseup，mousedown。

MouseEvent 派生自 UIEvent，UIEvent 派生自 Event。虽然 MouseEvent.initMouseEvent() 方法保持向后兼容性，但是应该使用 MouseEvent() 构造函数创建一个 MouseEvent 对象。

属性 读写 描述 DOM
altKey 只读 当鼠标事件触发的时，如果alt键被按下，返回true; 2
button 只读 当鼠标事件触发的时，如果鼠标按钮被按下（如果有的话），将会返回一个数值。 2
buttons 只读 当鼠标事件触发的时，如果多个鼠标按钮被按下（如果有的话），将会返回一个或者多个代表鼠标按钮的数字。 2
clientX 只读 鼠标指针在点击元素（DOM）中的X坐标。 2
clientY 只读 鼠标指针在点击元素（DOM）中的Y坐标。 2
ctrlKey 只读 当鼠标事件触发时，如果control键被按下，则返回true； 2
metaKey 只读 当鼠标事件触发时，如果meta键被按下，则返回true； 2
movementX 只读 鼠标指针相对于最后mousemove事件位置的X坐标。 2
movementY 只读 鼠标指针相对于最后mousemove事件位置的Y坐标。 2
offsetX 只读 鼠标指针相对于目标节点内边位置的X坐标 实验性的 API
offsetY 只读 鼠标指针相对于目标节点内边位置的Y坐标 实验性的 API
pageX 只读 鼠标指针相对于整个文档的X坐标； 实验性的 API
pageY 只读 鼠标指针相对于整个文档的Y坐标； 实验性的 API
region 只读 返回被点击事件影响的点击区域的id，如果没有区域被影响则返回null。 2
relatedTarget 只读 鼠标事件的次要目标（如果有的话） 2
screenX 只读 鼠标指针相对于全局（屏幕）的X坐标； 2
screenY 只读 鼠标指针相对于全局（屏幕）的Y坐标； 2
shiftKey 只读 当鼠标事件触发时，如果shift键被按下，则返回true； 2
which 只读 当鼠标事件触发时，表示被按下的按钮。 此API尚未标准化
mozPressure 只读 点击事件发生时施加在触摸屏或者平板设备的压力量。这个数值在0.0（最小压力）和1.0（最大压力）之间。 此API尚未标准化
mozInputSource 只读 生成事件的类型（若干MOZ_SOURCE_*常量如下列出）。可通过该属性获知鼠标事件是否由真实鼠标设备触发，亦或通过触摸事件触发（这可能影响处理坐标事件时的精确程度）。 此API尚未标准化
webkitForce 只读 点击时施加的压力量。 此API尚未标准化
x 只读 clientX的别名。 实验性的 API
y 只读 clientY的别名。 实验性的 API

一、clientX、clientY

点击位置距离当前body可视区域的x，y坐标

二、pageX、pageY

对于整个页面来说，包括了被卷去的body部分的长度

三、screenX、screenY

点击位置距离当前电脑屏幕的x，y坐标

四、offsetX、offsetY

相对于带有定位的父盒子的x，y坐标

五、x、y

和screenX、screenY一样
