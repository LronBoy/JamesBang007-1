选择器——获取节点 
    document.getElementById(元素id); //通过元素id获取节点
    document.getElementsByTagName(元素标签); //通过元素标签获取节点
    document.getElementsByName(元素name); //通过元素name获取节点

    document.getElementsByClassName(class类名); //通过元素class类名获取节点  IE9+
    document.querySelector(CSS选择器); //类似jQuery，匹配指定CSS选择器的一个元素   IE8+
    document.querySelectorAll(CSS选择器); //HTML5中引入的新方法，匹配的CSS选择器的所有元素节点

For循环
    for (语句1; 语句2; 语句3)
    {
              被执行的代码块;
    }

语句1 在循环（代码块）开始前执行
语句2 定义运行循环的条件
语句3 在循环已被执行之后执行

例1：说明for循环的执行顺序

    for(var i=0; i<3; i++){
       console.log(i);
    }

    var i = 0;
    for ( ; i<3; ) {
      console.log(1);
      i++;
    }

    1) var i=0;
    2) i<3; 关键！！
    3) 括号里面的所有代码
    4) i++
